ArrayList - реализация изменяемого массива интерфейса List, часть Collection Framework, который отвечает за список (или динамический массив), расположенный в пакете java.utils. Этот класс реализует все необязательные операции со списком и предоставляет методы управления размером массива, который используется для хранения списка.

В основе лежит идея динамического массива. А именно, возможность добавлять и удалять элементы, при этом будет увеличиваться или уменьшаться по мере необходимости.


ArrayList хранит только ссылочные типы, любые объекты, включая сторонние классы. Для хранения примитивных типов данных используются классы обертки.


Конструкторы ArrayList:
Пустой с начальной емкостью внутреннего массива = 10
ArrayList<String> list = new ArrayList<>()

ArrayList(Collection <? extends E) c)
Конструктор принимает другую коллекцию, создавая новый массив с элементами переданной коллекции
ArrayList<String> list2 = new ArrayLIst<>(list)

ArrayList(int initialCapacity)
В качестве параметра выступает значение начального размера внутреннего массива.
ArrayList<String> list2 = new ArrayList<>(10000);


Методы
add(E e) - добавление нового элемента в конец списка

add(int index, E element) - добавление нового элемента в позицию index. При добавлении все элементы справа от индекса сдвигаются

addAll(Collection <? extend E> collection) - добавление всех элементов коллекции collection в список в порядке их расположения в collection.

addAll(int index, Collection <? extends E> collection) - добавление всех элементов collection в список начиная с индекса index

clear() - удаление всех элементов из списка

clone() - возвращение объекта-копии массива

contains() - проверка на наличие

ensureCapacity(int minCapacity) - увеличение размера внутреннего массива, чтобы в него поместилось количество элементов, переданных в minCapacity.

get(int index) - возвращает элемент, который указан в данной позиции

indexOf(Object o) - индекс первого вхождения элемента в списке. Если элемента не существует в списке, метод вернет -1.

isEmpty() - возвращает true, если список пустой

iterator - возвращает итератор для списка для последующего использования в цикле или при любой другой обработке

lastIndexOf(Object o) - возвращается индекс последнего элемента в списке

remove(int index) - удаление элемента в указанной позиции индекса. После удаления все элементы справа сдвигаются влево

remove(Object o) - метод удаляет из списка переданный элемент о.

removeAll() - принимает коллекцию элементов, которая будет удалена из списка.

set(int index, E element) - замена элемента в указанной позиции index на переданный element

size() - узнать размер массива

sort(Comparator<? super E> c) - сортировка по заданному правилу

toArray() - превращение списка в фиксированный массив
