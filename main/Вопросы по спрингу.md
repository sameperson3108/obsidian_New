Какую задачу решает спринг бут?
Автоконфигурация

Зависимости, которые спринг создаёт и поддерживает?
context, web, security

Как поддерживать совместимость зависимостей?
Спринг бут это делает самостоятельно, разработчик просто должен использовать 1 версию спринг бута

Какие основные скоупы для бина в спринге?
singleton, prototype

Сколько раз bean prototype будет создан как часть бина singleton?
Ровно столько, сколько будет создан бин носитель со скопом singleton

При помощи каких аннотаций можно создать bean?
Bean, Component, Service, Controller, RestController, Repository

Какие способы конфигурирования бинов существуют?
xml configuration, java configuration, annotation based configuration

Каким образом спринг присваивает уникальное имя бину?
Если используется аннотация, то имя бина берётся на основе класса, над которым стоит данная аннотация
